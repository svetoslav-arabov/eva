📡 Radio Telescope Project

Building a working radio telescope and documenting the journey.

Project Goal

The goal of this project is to design and build a functional radio telescope capable of detecting natural radio emissions from space. This includes learning about antenna design, signal processing, software development, and radio astronomy. The long-term objective is to observe sources such as the Milky Way and other cosmic radio signals.
